🔥 Firebase Realtime DB — Sync ทุกอุปกรณ์
⚪ ยังไม่ได้เชื่อมต่อ Firebase
📌 วิธีตั้งค่า Firebase ครั้งแรก:
1. ไปที่ console.firebase.google.com → สร้างโปรเจกต์ใหม่
2. ไปที่ Realtime Database → Create database → เลือก "Start in test mode"
3. ไปที่ Project Settings → Your apps → Web app → Copy config
4. วาง apiKey, projectId และ Database URL ด้านบน → กด "เชื่อมต่อ"
5. แชร์ลิงก์ไฟล์นี้ให้พนักงาน — ทุกคนจะเห็นข้อมูลเดียวกัน real-time ✓

📌 วิธีใช้: สร้าง Google Apps Script → วาง Code.gs จากไฟล์ที่ให้ → Deploy → Copy URL มาวางที่นี่
